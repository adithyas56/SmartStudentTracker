# Smart Student Tracker

A Java-based Student Management System using JDBC and MySQL.

## Features
- Add Student
- View Students
- Update Student
- Delete Student
- CLI Menu System
- DAO Design Pattern

## Technologies Used
- Java
- JDBC
- MySQL
- IntelliJ IDEA

## How to Run
1. Clone the repository
2. Import into IntelliJ
3. Configure MySQL database
4. Run MainApp.java
